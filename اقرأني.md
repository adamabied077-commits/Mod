# مود MP40 — Fabric 1.21.11

## ملاحظة مهمة بصراحة
أنا كتبت ليك الكود الكامل ديال المود (100% جاهز وخدام)، ولكن ما نقدرش نصايب ليك ملف `.jar` جاهز
مباشرة من عندي، لأن صايبو المود كيحتاج تحميل Minecraft + Fabric API + Loom من الإنترنت،
والبيئة ديالي هنا ماعندهاش connection بالإنترنت. الحل: خاصك تدير "build" وحدة بسيطة عندك،
وغادي تولي عندك ملف `.jar` رهيب تحطو فـ `mods` وخدام عادي، بلا ما تكتب ولا سطر كود.

## شنو كاين فهاد الملف
- سلاح **MP40** كيضرب أوتوماتيك (full-auto) طول ما ضاغط الزر الليمن
- ذخيرة خاصة **MP40 Bullet**
- كرافتينغ للسلاح وللذخيرة
- صور بسيطة (pixel art) للسلاح والرصاصة

## الخطوات (10 دقايق، غير مرة وحدة)

### 1) حمل القالب الرسمي ديال Fabric
سير لهاد الرابط: **https://fabricmc.net/develop/template/**
- Minecraft Version: `1.21.11`
- Mod Name: `MP40 Weapon Mod`
- Package Name: `com.mp40mod`
- Mod ID: `mp40mod`
- خلي الباقي default، ودير "Generate" و"Download" باش يهبطلك zip فيه `gradlew` جاهز.

### 2) بدل الملفات
- فك (extract) الـ zip اللي هبطتيه من القالب الرسمي
- فك زيد هاد الملف اللي عطيتك (`mp40mod.zip`)
- **دخل** المجلدات ديال `mp40mod.zip` فوق المجلدات ديال القالب الرسمي، وقبل "Replace/Overwrite" فـ كل حاجة كتشابه (خاصة `build.gradle`, `gradle.properties`, `settings.gradle`, `fabric.mod.json`, و`src/`)

### 3) ابني (Build)
افتح الـ Terminal فداخل المجلد ديال المشروع، وكتب:

**على Mac/Linux:**
```
./gradlew build
```

**على Windows:**
```
gradlew.bat build
```

⚠️ خاصك تكون مربوط بالإنترنت أول مرة (غادي يحمل Minecraft + Fabric API تلقائيًا، حتى 5-10 دقايق).
⚠️ خاصك يكون عندك **Java 21** مثبت (تأكد بـ `java -version`).

### 4) لقا الـ jar وحطو فـ mods
من بعد ما يكمل الـ build بنجاح، غادي تلقى الملف هنا:
```
build/libs/mp40mod-1.0.0.jar
```
هاد الملف هو اللي خاصك تحطو فـ `.minecraft/mods` (خاصك عندك Fabric Loader 0.18.1+ و Fabric API 0.141.5+1.21.11 مثبتين فـ نفس الفولدر ديال mods).

## كيفاش تلعب
1. صايب السلاح فـ crafting table (شوف الوصفة فـ `data/mp40mod/recipe/mp40.json`)
2. صايب شوية رصاص (mp40_bullet)
3. حط الرصاص فـ الجيب ديالك، وحط السلاح فـ يدك
4. اضغط وخلي مضغوط الزر الليمن (Right Click) → غادي يضرب أوتوماتيك حتى تلقي الزر ولا تسالي الذخيرة

## بغيتي تبدل شي حاجة؟
- سرعة الضرب: `FIRE_INTERVAL_TICKS` فـ `Mp40Item.java`
- الدمّاج: `BULLET_DAMAGE`
- سرعة الرصاصة: `BULLET_VELOCITY`
- الصور: `assets/mp40mod/textures/item/mp40.png` و `mp40_bullet.png` (16x16 px)

قوليا إلا بغيتي نزيد: صوت خاص بلاصة صوت الـ crossbow، موديل 3D حقيقي، سكوب/زوم، mode شوت وحدة وحدة، ولا حتى نظام reload.
